## Default Permission

Allows the shared Lilia runtime to update the native window backdrop.

#### This default permission set includes the following:

- `allow-set-window-backdrop`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`lilia:allow-set-window-backdrop`

</td>
<td>

Enables the set_window_backdrop command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`lilia:deny-set-window-backdrop`

</td>
<td>

</td>
</tr>
</table>
